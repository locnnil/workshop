# Workshop v0.1.10 release notes

## 21 February 2025

These release notes cover new features and changes in Workshop v0.1.10.

## Requirements and Compatibility

Normally, Workshop relies on Snap and LXD in its operation:

- For installation details, see the
  [Tutorial](https://canonical-workshop.readthedocs-hosted.com/stable/tutorial/).
- For additional development requirements, see the [Contribution
  Guide](https://canonical-workshop.readthedocs-hosted.com/stable/contributing/).

## What’s New in Workshop v0.1.10

This is mostly an intermediate bugfix release. The fixes include:

- Shell completion behavior, which wasn't working properly
- Improved ``apt`` experience and settings that reduce the need for manual
  configuration by users
- Fixed desktop interface behavior with X11
- Added configuration for the ``errcheck`` linter
- Adjusted the release workflow to update JSON schemas in documentation
