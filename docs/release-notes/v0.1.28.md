# Workshop v0.1.28 release notes

## 8 November 2025

These release notes cover new features and changes in Workshop v0.1.28.

## Requirements and compatibility

Workshop relies on Snap and LXD:

- See the [Tutorial](https://canonical-workshop.readthedocs-hosted.com/stable/tutorial/) for setup instructions.
- Refer to the [Contribution Guide](https://canonical-workshop.readthedocs-hosted.com/stable/contributing/) for development prerequisites.

## What's new in Workshop v0.1.28

This release improves the architecture documentation, realigns our docs test
matrix, and makes LXD base image lookups faster.

### Cached base image metadata

`GetBase` now tells LXD where to cache the large base-image metadata JSON, so
subsequent lookups reuse the file for an hour. This trims repeated downloads
and speeds up workshop launches that reuse the same Ubuntu bases.

### Documentation and documentation tests

The architecture guide now includes dedicated "Components" and "Runtime
behavior" sections. Information about `workshopd`, the LXD backend, interface
management, and data layout moved from other documentation sections into these
focused chapters for clearer understanding.

Documentation tests now match the site structure. The single tutorial test was
split into four part-specific checks; we added coverage for Git and debugging
how-tos; and the harness supports both real-store and mock-store modes.
