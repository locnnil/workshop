# Workshop v0.1.27 release notes

## 22 October 2025

These release notes cover new features and changes in Workshop v0.1.27.

## Requirements and compatibility

Workshop relies on Snap and LXD:

- See the [Tutorial](https://canonical-workshop.readthedocs-hosted.com/stable/tutorial/) for setup instructions.
- Refer to the [Contribution Guide](https://canonical-workshop.readthedocs-hosted.com/stable/contributing/) for development prerequisites.

## What's new in Workshop v0.1.27

This release improves compatibility with recent Ubuntu releases and kernels,
addressing systemd integration and container launch issues.

### Use systemctl instead of sudo

Workshop now uses `systemctl --machine` instead of `sudo -u` when querying user
environment variables. This change accommodates Ubuntu's transition to `sudo-rs`
in the latest release for Workshop to work correctly on systems using the new
`sudo` implementation.

### Workaround for Ubuntu 22.04 containers on recent kernels

We added a workaround for [snapd bug #2127244](https://bugs.launchpad.net/snapd/+bug/2127244),
which breaks LXD with snapd 2.71 and Ubuntu 22.04 on 6.14+ kernels.
This enables Workshop to run while the upstream is being fixed.
