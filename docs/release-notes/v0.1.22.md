# Workshop v0.1.22 release notes

## 15 August 2025

These release notes cover new features and changes in Workshop v0.1.22.

## Requirements and compatibility

Workshop relies on Snap and LXD:

- See the
  [Tutorial](https://canonical-workshop.readthedocs-hosted.com/stable/tutorial/)
  for setup instructions.
- Refer to the [Contribution
  Guide](https://canonical-workshop.readthedocs-hosted.com/stable/contributing/)
  for development prerequisites.

## What’s new in Workshop v0.1.22

### Support `sdkcraft try`

After running `sdkcraft try` in an SDKcraft project, the local user can use the SDK(s) it built
by adding `- name: try-<SDK>` to a workshop definition.
