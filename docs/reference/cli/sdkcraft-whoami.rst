sdkcraft whoami
---------------

.. @artefact sdkcraft whoami

Display login information

.. rubric:: Usage

.. code-block:: console

   $ sdkcraft whoami

.. rubric:: Description


Display information about the currently authenticated user.


.. rubric:: Examples


Show current login:

.. code-block:: console

   $ sdkcraft whoami
