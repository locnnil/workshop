workshop hack
-------------

Edit hack SDK

Synopsis
~~~~~~~~

Edit hack SDK

.. code-block:: console

   workshop hack [--drop|--restore] <WORKSHOP> [setup-base|save-save|restore-state|check-health] [flags]


Options
~~~~~~~

.. code-block:: console

      --drop      Drop hack SDK from the workshop.
      --restore   Restore the dropped hack SDK for the workshop.
